integer function isize (   NUM)

THIS FUNCTION COMPUTES AND RETURNS THE NUMBER OF CHARACTERS NEEDED TO ENCODE THE INPUT INTEGER NUM AS A STRING.

IT DOES NOT ACTUALLY ENCODE THE STRING BUT RATHER ONLY FIGURES OUT THE REQUIRED SIZE. NUM MUST BE AN INTEGER IN THE RANGE OF 0 TO 99999.

PROGRAM HISTORY LOG: 2009-03-23 J. ATOR – ORIGINAL AUTHOR 2016-02-12 J. ATOR – MODIFIED FOR CRAYFTN COMPATIBILITY

USAGE: CALL ISIZE ( NUM ) INPUT ARGUMENT LIST: NUM - INTEGER: NUMBER TO BE ENCODED

OUTPUT ARGUMENT LIST: ISIZE - INTEGER: NUMBER OF CHARACTERS NECESSARY TO ENCODE NUM AS A STRING

REMARKS: THIS ROUTINE CALLS: BORT THIS ROUTINE IS CALLED BY: MTFNAM UFBDMP UFDUMP Normally not called by any application programs but it could be.
