THIS SUBROUTINE BUILDS THE ENTIRE JUMP/LINK TREE (I.E., INCLUDING RECURSIVELY RESOLVING ALL "CHILD" MNEMONICS) FOR A TABLE A MNEMONIC (NEMO) WITHIN THE INTERNAL JUMP/LINK TABLE.

USAGE: CALL TABSUB (LUN, NEMO) INPUT ARGUMENT LIST: LUN - INTEGER: I/O STREAM INDEX INTO INTERNAL MEMORY ARRAYS NEMO - CHARACTER*8: TABLE A MNEMONIC

REMARKS:

EXAMPLE SHOWING CONTENTS OF INTERNAL JUMP/LINK TABLE (WITHIN MODULE TABLES):

INTEGER MAXTAB = maximum number of jump/link table entries

INTEGER NTAB = actual number of jump/link table entries currently in use

For I = 1, NTAB:

CHARACTER*10 TAG(I) = mnemonic

CHARACTER*3 TYP(I) = mnemonic type indicator: "SUB" if TAG(I) is a Table A mnemonic "SEQ" if TAG(I) is a Table D mnemonic using either short (i.e. 1-bit) delayed replication, F=1 regular (i.e. non-delayed) replication, or no replication at all "RPC" if TAG(I) is a Table D mnemonic using either medium (i.e. 8-bit) delayed replication or long (i.e. 16-bit) delayed replication "RPS" if TAG(I) is a Table D mnemonic using medium (i.e. 8-bit) delayed replication in a stack context "DRB" if TAG(I) denotes the short (i.e. 1-bit) delayed replication of a Table D mnemonic (which would then itself have its own separate entry in the jump/link table with a corresponding TAG value of "SEQ") "DRP" if TAG(I) denotes either the medium (i.e. 8-bit) or long (i.e. 16-bit) delayed replication of a Table D mnemonic (which would then itself have its own separate entry in the jump/link table with a corresponding TAG value of "RPC") "DRS" if TAG(I) denotes the medium (i.e. 8-bit) delayed replication, in a stack context, of a Table D mnemonic (which would then itself have its own separate entry in the jump/link table with a corresponding TAG value of "RPS") "REP" if TAG(I) denotes the F=1 regular (i.e. non-delayed) replication of a Table D mnemonic (which would then itself have its own separate entry in the jump/link table with a corresponding TAG value of "SEQ") "CHR" if TAG(I) is a Table B mnemonic with units "CCITT IA5" "NUM" if TAG(I) is a Table B mnemonic with any units other than "CCITT IA5"

INTEGER JMPB(I):

IF ( TYP(I) = "SUB" ) THEN JMPB(I) = 0 ELSE IF ( ( TYP(I) = "SEQ" and TAG(I) uses either short (i.e. 1-bit) delayed replication or F=1 regular (i.e. non-delayed) replication ) OR ( TYP(I) = "RPC" ) ) THEN JMPB(I) = the index of the jump/link table entry denoting the replication of TAG(I) ELSE JMPB(I) = the index of the jump/link table entry for the Table A or Table D mnemonic of which TAG(I) is a child END IF

INTEGER JUMP(I):

IF ( ( TYP(I) = "CHR" ) OR ( TYP(I) = "NUM" ) ) THEN JUMP(I) = 0 ELSE IF ( ( TYP(I) = "DRB" ) OR ( TYP(I) = "DRP" ) OR ( TYP(I) = "REP" ) ) THEN JUMP(I) = the index of the jump/link table entry for the Table D mnemonic whose replication is denoted by TAG(I) ELSE JUMP(I) = the index of the jump/link table entry for the Table B or Table D mnemonic which, sequentially, is the first child of TAG(I) END IF

INTEGER LINK(I):

IF ( ( TYP(I) = "SEQ" and TAG(I) uses either short (i.e. 1-bit) delayed replication or F=1 regular (i.e. non- delayed) replication ) OR ( TYP(I) = "SUB" ) OR ( TYP(I) = "RPC" ) ) THEN LINK(I) = 0 ELSE IF ( TAG(I) is, sequentially, the last child Table B or Table D mnemonic of the parent Table A or Table D mnemonic indexed by JMPB(I) ) THEN LINK(I) = 0 ELSE LINK(I) = the index of the jump/link table entry for the Table B or Table D mnemonic which, sequentially, is the next (i.e. following TAG(I)) child mnemonic of the parent Table A or Table D mnemonic indexed by JMPB(I) END IF

INTEGER IBT(I):

IF ( ( TYP(I) = "CHR" ) OR ( TYP(I) = "NUM" ) ) THEN IBT(I) = bit width of Table B mnemonic TAG(I) ELSE IF ( ( TYP(I) = "DRB" ) OR ( TYP(I) = "DRP" ) ) THEN IBT(I) = bit width of delayed descriptor replication factor (i.e. 1, 8, or 16, depending on the replication scheme denoted by TAG(I)) ELSE IBT(I) = 0 END IF

INTEGER IRF(I):

IF ( TYP(I) = "NUM" ) THEN IRF(I) = reference value of Table B mnemonic TAG(I) ELSE IF ( TYP(I) = "REP" ) THEN IRF(I) = number of F=1 regular (i.e. non-delayed) replications of Table D mnemonic TAG(JUMP(I)) ELSE IRF(I) = 0 END IF

INTEGER ISC(I):

IF ( TYP(I) = "NUM" ) THEN ISC(I) = scale factor of Table B mnemonic TAG(I) ELSE IF ( TYP(I) = "SUB" ) THEN ISC(I) = the index of the jump/link table entry which, sequentially, constitutes the last element of the jump/link tree for Table A mnemonic TAG(I) ELSE ISC(I) = 0 END IF

THE FOLLOWING VALUES ARE STORED WITHIN MODULE NRV203 BY THIS SUBROUTINE, FOR USE WITH ANY 2-03-YYY (CHANGE REFERENCE VALUE) OPERATORS PRESENT WITHIN THE ENTIRE JUMP/LINK TABLE:

NNRV = number of nodes in the jump/link table which contain new reference values (as defined using the 2-03 operator)

INODNRV(I=1,NNRV) = nodes within jump/link table which contain new reference values

NRV(I=1,NNRV) = new reference value corresponding to INODNRV(I)

TAGNRV(I=1,NNRV) = Table B mnemonic to which the new reference value in NRV(I) applies

ISNRV(I=1,NNRV) = start of node range in jump/link table, within which the new reference value defined by NRV(I) will be applied to all occurrences of TAGNRV(I)

IENRV(I=1,NNRV) = end of node range in jump/link table, within which the new reference value defined by NRV(I) will be applied to all occurrences of TAGNRV(I)

IBTNRV = number of bits in Section 4 occupied by each new reference value for the current 2-03 operator (if IBTNRV = 0, then no 2-03 operator is currently in scope)

IPFNRV = a number between 1 and NNRV, denoting the first entry within the above arrays which applies to the current Table A mnemonic NEMO (if IPFNRV = 0, then no 2-03 operators have been applied to NEMO)

THIS ROUTINE CALLS: BORT INCTAB IOKOPER NEMTAB NEMTBD TABENT THIS ROUTINE IS CALLED BY: MAKESTAB Normally not called by any application programs.
