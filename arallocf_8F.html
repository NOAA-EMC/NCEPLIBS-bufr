subroutine arallocf

IF DYNAMIC MEMORY ALLOCATION IS BEING USED, THIS ROUTINE IS CALLED DURING THE FIRST CALL TO BUFR ARCHIVE LIBRARY SUBROUTINE OPENBF TO DYNAMICALLY ALLOCATE MEMORY FOR ALL REQUIRED FORTRAN LANGUAGE ARRAYS.

THESE ARRAYS ARE SIZED USING VALUES INPUT DURING ONE OR MORE PREVIOUS CALLS TO BUFR ARCHIVE LIBRARY SUBROUTINE ISETPRM, OR ELSE USING ONE OR MORE DEFAULT VALUES SPECIFIED IN MODULE FILES IF ISETPRM IS NEVER CALLED FOR THOSE PARTICULAR SIZE VALUES.

MEMORY ALLOCATED WITHIN THIS ROUTINE CAN BE FREED VIA A USER CALL TO BUFR ARCHIVE LIBRARY ROUTINE ARDLLOCF (IN CASES WHERE THE APPLICATION PROGRAM MAY WISH TO MOVE ON TO OTHER TASKS NOT REQUIRING ANY FURTHER CALLS TO BUFR ARCHIVE LIBRARY ROUTINES), OR ELSE IT WILL BE FREED AUTOMATICALLY WHEN THE APPLICATION PROGRAM TERMINATES.

USAGE: CALL ARALLOCF

REMARKS: THIS ROUTINE CALLS: BORT ERRWRT THIS ROUTINE IS CALLED BY: OPENBF Normally not called by any application programs.
