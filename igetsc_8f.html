function igetsc (   LUNIT)

THIS FUNCTION RETURNS ANY STATUS CODE THAT WAS INTERNALLY SET WITHIN THE BUFR ARCHIVE LIBRARY SOFTWARE FOR A GIVEN LOGICAL UNIT NUMBER.

PROGRAM HISTORY LOG: 2010-05-11 J. ATOR – ORIGINAL AUTHOR 2014-12-10 J. ATOR – USE MODULES INSTEAD OF COMMON BLOCKS

USAGE: IGETSC (LUNIT) INPUT ARGUMENT LIST: LUNIT - INTEGER: FORTRAN LOGICAL UNIT NUMBER FOR BUFR FILE

OUTPUT ARGUMENT LIST: IGETSC - INTEGER: STATUS CODE FOR LUNIT: 0 = no problems noted with LUNIT -1 = unable to position LUNIT for appending, possibly due to an incomplete BUFR message at the end of the file

REMARKS: THIS ROUTINE CALLS: BORT STATUS THIS ROUTINE IS CALLED BY: None Normally called only by application programs.
