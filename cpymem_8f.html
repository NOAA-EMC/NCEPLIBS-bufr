subroutine cpymem (   LUNOT)

THIS SUBROUTINE COPIES A BUFR MESSAGE, INTACT, FROM INTERNAL MEMORY, STORED VIA A PREVIOUS CALL TO BUFR ARCHIVE LIBRARY SUBROUTINE UFBMEM, TO LOGICAL UNIT LUNOT, OPENED FOR OUTPUT VIA A PREVIOUS CALL TO BUFR ARCHIVE LIBRARY SUBROUTINE OPENBF.

THE MESSAGE COPIED FROM INTERNAL MEMORY WILL BE THE ONE MOST RECENTLY READ INTO THE MESSAGE BUFFER (ARRAY MBAY IN MODULE BITBUF) USING BUFR ARCHIVE LIBRARY SUBROUTINE RDMEMM OR READMM. THE OUTPUT FILE MUST HAVE NO CURENTLY OPEN MESSAGES. ALSO, THE INTERNAL BUFR TABLES ASSOCIATED WITH THE INPUT MESSAGE MUST BE IDENTICAL TO THE BUFR TABLES USED TO OPEN LUNOT TO THE BUFR INTERFACE. THIS SUBROUTINE IS SIMILAR TO BUFR ARCHIVE LIBRARY SUBROUTINE COPYMG EXCEPT THE INPUT MESSAGE IS FROM INTERNAL MEMORY NOT FROM A PHYSICAL BUFR FILE.
