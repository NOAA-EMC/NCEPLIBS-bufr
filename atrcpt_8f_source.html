4 C> THIS SUBROUTINE READS AN INPUT BUFR MESSAGE, APPENDS THE
5 C> TANK RECEIPT TIME TO SECTION 1, AND WRITES THE RESULT TO A NEW BUFR
6 C> MESSAGE FOR OUTPUT. THE TANK RECEIPT TIME MUST HAVE BEEN SPECIFIED
7 C> VIA A PREVIOUS CALL TO BUFR ARCHIVE LIBRARY SUBROUTINE STRCPT. THE
8 C> OUTPUT MESSAGE WILL BE SLIGHTLY LONGER THAN THE INPUT MESSAGE, SO
9 C> THE USER MUST ALLOW FOR ENOUGH SPACE WITHIN THE OUTPUT ARRAY.
10 C>
11 C> PROGRAM HISTORY LOG:
12 C> 2009-03-23 J. ATOR -- ORIGINAL AUTHOR
13 C>
14 C> USAGE: CALL ATRCPT (MSGIN, LMSGOT, MSGOT)
15 C> INPUT ARGUMENT LIST:
16 C> MSGIN - INTEGER: *-WORD ARRAY CONTAINING BUFR MESSAGE
17 C> LMSGOT - INTEGER: DIMENSIONED SIZE (IN INTEGER WORDS) OF MSGOT;
18 C> USED BY THE SUBROUTINE TO ENSURE THAT IT DOES NOT
19 C> OVERFLOW THE MSGOT ARRAY
20 C>
21 C> OUTPUT ARGUMENT LIST:
22 C> MSGOT - INTEGER: *-WORD ARRAY CONTAINING INPUT BUFR MESSAGE
23 C> WITH TANK RECEIPT TIME APPENDED TO SECTION 1
24 C>
25 C> REMARKS:
26 C> MSGIN AND MSGOT MUST BE SEPARATE ARRAYS.
27 C>
28 C> THIS ROUTINE CALLS: BORT GETLENS IUPBS01 MVB
29 C> PKB
30 C> THIS ROUTINE IS CALLED BY: MSGWRT
