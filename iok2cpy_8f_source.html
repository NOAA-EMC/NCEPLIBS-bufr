4 C> THIS FUNCTION DETERMINES WHETHER A MESSAGE, OR A SUBSET
5 C> FROM A MESSAGE, CAN BE COPIED FROM LOGICAL UNIT IOLUN(LUI) TO
6 C> LOGICAL UNIT IOLUN(LUO). THE DECISION IS BASED ON WHETHER THE
7 C> EXACT SAME DEFINITION FOR THE GIVEN MESSAGE TYPE APPEARS WITHIN
8 C> THE DICTIONARY TABLE INFORMATION FOR BOTH LOGICAL UNITS. NOTE THAT
9 C> IT IS POSSIBLE FOR A MESSAGE TYPE TO BE IDENTICALLY DEFINED FOR TWO
10 C> DIFFERENT LOGICAL UNITS EVEN IF THE UNITS THEMSELVES DON'T SHARE
11 C> THE EXACT SAME FULL SET OF DICTIONARY TABLES.
12 C>
