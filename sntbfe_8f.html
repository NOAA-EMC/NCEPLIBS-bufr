THIS SUBROUTINE READS A COMPLETE ENTRY (CORRESPONDING TO THE INPUT FXY NUMBER) FROM AN ASCII MASTER CODE/FLAG TABLE AND STORES THE OUTPUT INTO AN INTERNAL MEMORY STRUCTURE.

PROGRAM HISTORY LOG: 2017-11-02 J. ATOR – ORIGINAL AUTHOR

USAGE: CALL SNTBFE ( LUNT, IFXYN, LINE ) INPUT ARGUMENT LIST: LUNT - INTEGER: FORTRAN LOGICAL UNIT NUMBER OF ASCII FILE CONTAINING MASTER CODE/FLAG TABLE INFORMATION IFXYN - INTEGER: BIT-WISE REPRESENTATION OF FXY NUMBER LINE - CHARACTER*(*): FIRST LINE OF TABLE ENTRY

REMARKS: THIS ROUTINE CALLS: ADN30 BORT2 IFXY IGETFXY IGETNTBL JSTCHR PARSTR STRNUM STRTBFE THIS ROUTINE IS CALLED BY: RDMTBF Normally not called by any application programs.
