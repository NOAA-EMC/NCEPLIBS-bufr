C C SUBPROGRAM: CMPSTIA1 C PRGMMR: ATOR ORG: NCEP DATE: 2017-11-13 C C ABSTRACT: THIS ROUTINE DEFINES A COMPARISON BETWEEN TWO ENTRIES IN C THE INTERNAL MEMORY STRUCTURE USED FOR STORING ASCII MASTER C CODE/FLAG TABLE INFORMATION.

THE COMPARISON IS USED BY THE C BINARY SEARCH FUNCTIONS QSORT AND BSEARCH, AND IT DIFFERS FROM THE C LOGIC IN ROUTINE CMPSTIA2 BECAUSE IT COMPARES ALL OF THE IFFXY, C IFVAL, IFFXYD and IFVALD COMPONENTS OF THE STRUCTURE, WHEREAS C CMPSTIA2 ONLY COMPARES THE IFFXYN AND IFVAL COMPONENTS.
